# Schriftarten – Herkunft und Lizenz

Alle in diesem Verzeichnis eingebundenen Schriftdateien stammen unverändert von Google Fonts
(`fonts.gstatic.com`, `latin`-Subset) und stehen unter der SIL Open Font License 1.1 (OFL).
Die jeweilige Lizenzdatei mit dem vollständigen Lizenztext und dem Copyright-Hinweis liegt
diesem Verzeichnis bei.

| Schriftart | Datei | Designer / Projekt | Lizenz | Lizenztext |
|---|---|---|---|---|
| Archivo | `archivo-400.woff2` | Omnibus-Type ([Archivo-Projekt](https://github.com/Omnibus-Type/Archivo)) | SIL OFL 1.1 | [`archivo-OFL.txt`](archivo-OFL.txt) |
| Space Grotesk | `space-grotesk-var.woff2` | Florian Karsten ([Space-Grotesk-Projekt](https://github.com/floriankarsten/space-grotesk)) | SIL OFL 1.1 | [`space-grotesk-OFL.txt`](space-grotesk-OFL.txt) |
| Space Mono | `space-mono-400.woff2`, `space-mono-700.woff2` | Colophon Foundry ([Space-Mono-Projekt](https://github.com/googlefonts/spacemono)) | SIL OFL 1.1 | [`space-mono-OFL.txt`](space-mono-OFL.txt) |

Die Schriftdateien selbst wurden nicht verändert (lediglich der Bezug von Google-Servern auf
lokales Hosting umgestellt). Die OFL erlaubt Bündelung und Weitergabe unter der Bedingung,
dass Copyright-Hinweis und Lizenztext beiliegen – das ist mit den drei `*-OFL.txt`-Dateien
erfüllt.
